# \# دليل استخدام نظام إدارة المواعيد

# 

# \## نظرة عامة

# 

# نظام إدارة المواعيد هو تطبيق ويب متكامل مصمم لإدارة حجوزات العيادات الطبية بكفاءة عالية. يدعم النظام ثلاثة أنواع من المستخدمين بصلاحيات مختلفة لضمان سير العمل بسلاسة.

# 

# \## أنواع المستخدمين والصلاحيات

# 

# \### 1. الأدمن (Admin)

# المسؤول الرئيسي عن النظام ويمتلك صلاحيات كاملة:

# \- إدارة جميع المواعيد (عرض، تعديل، حذف)

# \- إدارة قائمة الأطباء (إضافة، تعديل، تفعيل/تعطيل)

# \- إدارة المستخدمين وتعيين الصلاحيات

# \- عرض إحصائيات شاملة لجميع الموظفين

# \- مراقبة أداء الفريق

# 

# \### 2. الكول سنتر (Call Center)

# موظف استقبال المكالمات وحجز المواعيد:

# \- إضافة حجوزات جديدة

# \- عرض وإدارة حجوزاته الشخصية فقط

# \- تعديل وحذف حجوزاته

# \- عرض إحصائياته الشخصية

# \- مشاهدة لوحة المتصدرين للتحفيز

# 

# \### 3. الرسبشن (Reception)

# موظف الاستقبال في العيادة:

# \- عرض جميع المواعيد

# \- تغيير حالة الموعد (مجدول، وصل، لم يصل، لم يرد)

# \- إضافة السعر للحجوزات عند وصول المريض

# \- فلترة المواعيد حسب التاريخ والحالة

# 

# \## البدء في استخدام النظام

# 

# \### تسجيل الدخول

# 1\. افتح رابط النظام في المتصفح

# 2\. انقر على زر "تسجيل الدخول"

# 3\. قم بتسجيل الدخول باستخدام حساب Manus الخاص بك

# 4\. سيتم توجيهك تلقائياً إلى لوحة التحكم المناسبة حسب صلاحيتك

# 

# \## دليل الأدمن

# 

# \### لوحة التحكم الرئيسية

# عند تسجيل الدخول كأدمن، ستشاهد:

# \- \*\*إحصائيات عامة\*\*: إجمالي المواعيد، مواعيد اليوم، عدد الأطباء، عدد الموظفين

# \- \*\*إحصائيات الموظفين\*\*: عرض أداء كل موظف (إجمالي الحجوزات وحجوزات اليوم)

# \- \*\*الإجراءات السريعة\*\*: أزرار للوصول السريع إلى إدارة المواعيد والأطباء والمستخدمين

# 

# \### إدارة الأطباء

# 1\. انقر على "إدارة الأطباء" من القائمة الجانبية

# 2\. لإضافة طبيب جديد:

# &nbsp;  - انقر على "إضافة طبيب جديد"

# &nbsp;  - أدخل اسم الطبيب والتخصص

# &nbsp;  - انقر "إضافة"

# 3\. لتعديل بيانات طبيب:

# &nbsp;  - انقر على أيقونة التعديل بجانب اسم الطبيب

# &nbsp;  - قم بتعديل البيانات

# &nbsp;  - يمكنك تفعيل أو تعطيل الطبيب باستخدام المفتاح

# &nbsp;  - انقر "حفظ التغييرات"

# 4\. لحذف طبيب:

# &nbsp;  - انقر على أيقونة الحذف

# &nbsp;  - أكد عملية الحذف

# 

# \### إدارة المستخدمين

# 1\. انقر على "إدارة المستخدمين" من القائمة

# 2\. ستظهر قائمة بجميع المستخدمين المسجلين

# 3\. لتغيير صلاحية مستخدم:

# &nbsp;  - اختر الصلاحية الجديدة من القائمة المنسدلة بجانب اسم المستخدم

# &nbsp;  - سيتم التحديث تلقائياً

# 

# \### إدارة المواعيد

# 1\. انقر على "إدارة المواعيد" من القائمة

# 2\. ستشاهد:

# &nbsp;  - إحصائيات المواعيد في الأعلى

# &nbsp;  - إحصائيات الموظفين

# &nbsp;  - جدول المواعيد مع إمكانية الفلترة حسب التاريخ

# 3\. لتعديل موعد:

# &nbsp;  - انقر على أيقونة التعديل

# &nbsp;  - قم بتعديل البيانات المطلوبة

# &nbsp;  - انقر "حفظ التغييرات"

# 4\. يمكنك تغيير حالة الموعد مباشرة من الجدول

# 5\. يمكنك حذف أي موعد بالنقر على أيقونة الحذف

# 

# \## دليل الكول سنتر

# 

# \### لوحة التحكم

# عند تسجيل الدخول كموظف كول سنتر، ستشاهد:

# \- \*\*إحصائياتك الشخصية\*\*: إجمالي حجوزاتك وحجوزات اليوم

# \- \*\*نموذج إضافة حجز جديد\*\*

# \- \*\*جدول حجوزاتك\*\* مقسم حسب التاريخ

# \- \*\*لوحة المتصدرين\*\*: عرض أفضل 3 موظفين للتحفيز

# 

# \### إضافة حجز جديد

# 1\. انقر على زر "حجز جديد"

# 2\. املأ البيانات المطلوبة:

# &nbsp;  - \*\*اسم المريض\*\* ورقم الهاتف (إلزامي)

# &nbsp;  - \*\*التاريخ والوقت\*\* (إلزامي)

# &nbsp;  - \*\*الطبيب\*\* من القائمة المنسدلة (إلزامي)

# &nbsp;  - \*\*نوع الحجز\*\*: كشف، متابعة، إلخ

# &nbsp;  - \*\*نوع المريض\*\*: عميل جديد أو قديم (إلزامي)

# &nbsp;  - \*\*مصدر الحجز\*\*: إعلانات، مكالمة، أو واتساب (إلزامي)

# &nbsp;  - \*\*ملاحظات\*\*: أي معلومات إضافية (اختياري)

# 3\. انقر "إضافة الموعد"

# 

# \### إدارة حجوزاتك

# \- استخدم حقل التاريخ لعرض حجوزات يوم معين

# \- يمكنك تعديل حجوزاتك بالنقر على أيقونة التعديل

# \- يمكنك حذف حجوزاتك بالنقر على أيقونة الحذف

# \- لا يمكنك رؤية أو تعديل حجوزات الموظفين الآخرين

# 

# \## دليل الرسبشن

# 

# \### لوحة التحكم

# عند تسجيل الدخول كموظف رسبشن، ستشاهد:

# \- \*\*إحصائيات اليوم\*\*: إجمالي المواعيد، عدد الوصول، المجدول، لم يصل، لم يرد

# \- \*\*جدول المواعيد\*\* مع إمكانية الفلترة

# 

# \### إدارة المواعيد

# 1\. استخدم الفلاتر في الأعلى:

# &nbsp;  - \*\*الحالة\*\*: اختر حالة معينة أو "الكل"

# &nbsp;  - \*\*التاريخ\*\*: اختر التاريخ المطلوب

# 2\. لتغيير حالة موعد:

# &nbsp;  - اختر الحالة الجديدة من القائمة المنسدلة في عمود "الحالة"

# &nbsp;  - سيتم التحديث تلقائياً

# 3\. لإضافة سعر لموعد:

# &nbsp;  - انقر على زر "إضافة" في عمود السعر

# &nbsp;  - أدخل المبلغ بالريال

# &nbsp;  - انقر "حفظ"

# 

# \### حالات المواعيد

# \- \*\*مجدول\*\*: الموعد محجوز وينتظر وصول المريض

# \- \*\*وصل\*\*: المريض وصل إلى العيادة

# \- \*\*لم يصل\*\*: المريض لم يحضر الموعد

# \- \*\*لم يرد\*\*: المريض لم يرد على المكالمة

# 

# \## نصائح وإرشادات

# 

# \### للأدمن

# \- راقب إحصائيات الموظفين بانتظام لتقييم الأداء

# \- تأكد من تحديث قائمة الأطباء عند إضافة أطباء جدد

# \- قم بتعيين الصلاحيات المناسبة لكل مستخدم

# 

# \### للكول سنتر

# \- تأكد من إدخال رقم الهاتف بشكل صحيح

# \- أضف ملاحظات مهمة في حقل الملاحظات

# \- راجع حجوزاتك قبل نهاية اليوم

# 

# \### للرسبشن

# \- قم بتحديث حالة المواعيد فور وصول المريض

# \- أضف السعر مباشرة عند الدفع

# \- استخدم الفلاتر لتسهيل البحث عن المواعيد

#

#
